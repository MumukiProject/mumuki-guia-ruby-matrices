No hay ninguna diferencia entre escribir una matriz como

```ruby
notas_por_alumno = [[8, 6, 9, 9], [6, 7, 7, 10], [8, 8, 8, 8]]
```

o como

```ruby
notas_por_alumno = [
  [8, 6, 9, 9],
  [6, 7, 7, 10],
  [8, 8, 8, 8]
]
```
